Credits

I would like to thank a number of people for their contributions to R2DToo. Thanks to:
  • Alan Garny for his TThreadedTimer high-resolution timer component which provides the timing for simulation timesteps and frame refreshes.
  • Grahame Grieve for his TxyGraph scientific graphing component which is used to plot the timeseries. (I find this component so useful I have foregone upgrading to newer versions of Borland C++Builder because of compatibility issues.)
  • Gordy Cowie for his TFastIMG bitmap image component (part of the FastLIB library) which is used for the view portal.
  • TMS Software for their TAdvSpreadGrid spreadsheet grid component which is used in the New dialog.
  • Brad Stowers for his TdfsVersionInfoResource file version information component which is used to retrieve the API version and display it in the program About box.
  • Fedor Koshevnikov, Igor Pavluk, and Serge Korolev, the authors of the RX Component Library, for a number of RX components used within the program.
  • Troels Skovmand Eriksen for his TProject versioning component which maintains a revision history for R2DToo.
  • Roldan Pozo for his stopwatch class (part of the Template Numerical Toolkit (TNT) library) which is used to control the relative processor time devoted to the simulation versus the display.
  • Markus Stephany for his TFloatEdit numerical edit box which is used in the Stop Conditions form.
  • Tamás Demjén for his TdtLabel hyperlinked label component which is used in the About boxes.
  • Jean-Michel Boucton for his TCheckGroupBox component which is in the Stop Conditions form.
  • Borland for their free command-line C++ compiler which allows programmers to write their own models for R2DToo.
  • Topica for providing the R2DToo Support mailing list.

R2DToo was financially supported by the National Science and Engineering Research Council of Canada (NSERC), 2000-2002.
